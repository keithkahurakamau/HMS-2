## 4. DATA PROCESSOR OBLIGATIONS

The Data Processor agrees to:

4.1 Process personal data only on documented instructions from the Data Controller, unless required by law.

4.2 Ensure that all staff authorised to process personal data have committed themselves to confidentiality or are under an appropriate statutory obligation of confidentiality.

4.3 Implement appropriate technical and organisational security measures including:
- Encryption of personal data in transit and at rest
- Access controls and authentication mechanisms
- Regular security testing and vulnerability assessments
- Incident detection and response capabilities

4.4 Not engage any sub-processor without prior written authorisation from the Data Controller.

4.5 Assist the Data Controller in fulfilling obligations relating to data subject rights (access, erasure, restriction, portability) within **5 business days** of receiving a request.

4.6 Delete or return all personal data to the Data Controller at the end of the service provision, as instructed by the Data Controller.

4.7 Make available all information necessary to demonstrate compliance with this Agreement and allow for audits conducted by the Data Controller or an appointed auditor.

---

## 5. BREACH NOTIFICATION

5.1 The Data Processor shall notify the Data Controller of any personal data breach **within 24 hours** of becoming aware of it.

5.2 The notification shall include: nature of the breach, categories and approximate number of records affected, likely consequences, and measures taken to address the breach.

5.3 The Data Processor shall cooperate fully with the Data Controller's breach investigation and ODPC notification obligations under KDPA Section 43.

---

## 6. DATA CONTROLLER OBLIGATIONS

The Data Controller shall:

6.1 Provide the Data Processor with clear documented instructions for all processing activities.

6.2 Ensure that there is a lawful basis for the transfer of personal data to the Data Processor.

6.3 Respond promptly to queries from the Data Processor regarding processing instructions.
